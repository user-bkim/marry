### Adding Your Photos

To add your own family photos to the gallery:

1. Create an `images` folder in this directory
2. Add your photos to the `images` folder
3. In `gallery.html` and `valentine.html`, replace the placeholder divs with actual image tags:

```html
<!-- Replace this: -->
<div class="photo-placeholder">
    <div class="placeholder-icon">üì∑</div>
    <p>Add your favorite photo here</p>
</div>

<!-- With this: -->
<img src="images/your-photo.jpg" alt="Description" class="gallery-photo">
```
